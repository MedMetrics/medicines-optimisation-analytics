# medicines-optimisation-analytics
Medicines optimisation analytics: prescribing variation across NHS primary care in England, using SQL, Power BI and SPC.

## How this was worked

Worked end to end as a delivery would be: intake brief, analysis
specification, data quality assessment, query library, data model,
report, and a stated limitations section.

## Data

OpenPrescribing.net (Bennett Institute for Applied Data Science,
University of Oxford). Practice-level, monthly, aggregated by BNF
presentation. No patient-level data is used or inferable.

## Governance

Raw extracts are excluded from version control. All data used is
openly published and contains no personal data.
